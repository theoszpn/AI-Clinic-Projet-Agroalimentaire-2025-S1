# Projet Semestriel AIGriTech - Data analysis
Projet tutoré au sein d'aivancity en groupe de 4.\
**Objectif :** déterminer à partir d'un dataset de 176600 lignes sur le marché agroalimentaire mondial (source FAO) quels pays sont les plus intéressants pour une expansion à l'international de l'activité de l'entreprise AIGriTech (simulation fictive). Focus sur les produits céréaliers.\
Le notebook jupyter contient le code complet de l'analyse ainsi que des visualisations graphiques.\
\
**Structure du notebook jupyter:**
1) EDA
2) Analyse des données agroalimentaires (Clustering, Scoring)
3) Analyse des données de population (Prédictions)
4) Modélisation et conclusion des analyses
